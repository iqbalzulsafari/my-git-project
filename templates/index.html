API Documentation

Overview

This API allows you to analyze images by sending a POST request to the endpoint. The analysis extracts any text present in the image using Azure's Computer Vision OCR service.

API Endpoint

The API is accessible at: http://localhost:3000/api/v1/analysis/

Request

To analyze an image, send a POST request with a JSON object containing a single field "url" pointing to the image to be analyzed. 

{
    "url": "https://raw.githubusercontent.com/iqbalzulsafari/my-git-project/main/sample-image.jpeg"
}

Response

If the request is successful, the API will return a JSON response with the extracted text from the image. 

{
    "text": "Extracted text from the image"
}

In case of an error, an error message will be returned: 

{
    "error": "An error occurred during image analysis"
}
